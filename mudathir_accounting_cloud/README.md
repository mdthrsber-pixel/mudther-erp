# نظام مدثر المحاسبي السحابي

منصة محاسبية تعمل على الجوال والكمبيوتر باستخدام:
- Next.js
- Supabase PostgreSQL
- صلاحيات المستخدمين
- فروع
- إيرادات يومية وشهرية
- بنك
- نقاط بيع
- مصروفات
- تقارير
- تصدير Excel / PDF

## التشغيل
1. افتح Supabase وأنشئ مشروع جديد.
2. انسخ محتوى `database/schema.sql` ونفذه داخل SQL Editor.
3. انسخ `.env.example` إلى `.env.local`.
4. ضع بيانات Supabase.
5. شغل:
```bash
npm install
npm run dev
```
