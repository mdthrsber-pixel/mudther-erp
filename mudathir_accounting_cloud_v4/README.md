# منصة مدثر المحاسبية السحابية V4

هذه نسخة أكثر اكتمالاً من النظام، تعمل على الجوال والكمبيوتر.

## الإضافات في V4
- لوحة دخول تجريبية.
- Dashboard متجاوب للجوال والكمبيوتر.
- CRUD للفروع.
- CRUD للإيرادات اليومية.
- CRUD للمصروفات.
- البنك.
- نقاط البيع.
- المستخدمين والصلاحيات.
- التقارير.
- تصدير Excel.
- تصدير PDF.
- ملف SQL كامل لقاعدة البيانات.
- ملف seed للبيانات التجريبية.
- PWA للتثبيت على الجوال.
- ملفات نشر Vercel و VPS.

## التشغيل
```bash
npm install
npm run dev
```

## ربط Supabase
1. أنشئ مشروع Supabase.
2. نفّذ `database/schema.sql`.
3. نفّذ `database/seed.sql`.
4. انسخ `.env.example` إلى `.env.local`.
5. ضع رابط Supabase و anon key.
